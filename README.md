# MinePass Discourse Plugin

Plugin for the [Discourse](http://discourse.org) forum system adding BBCodes for world banners.

## Usage

```
[minepass]WORLD_SUID[/minepass]
```

## Installation

* Add the plugin's repo url to your container's `app.yml` file

```yml
hooks:
  after_code:
    - exec:
        cd: $home/plugins
        cmd:
          - mkdir -p plugins
          - git clone https://github.com/discourse/docker_manager.git
          - git clone https://github.com/minepass/discourse-minepass.git
```

* Rebuild the container

```
cd /var/docker
git pull
./launcher rebuild app
```
